# 策略：QUALITY8 混合杠杆 + 放量1.5倍

## 定位

这是进攻型纸账户策略，用 LBank 日线数据模拟运行。策略只做多，但使用合约杠杆思路估算名义仓位、手续费、资金费率和爆仓风险。

## 纸账户

- 初始资金：2000 USDT
- 平台数据：LBank 日线
- 方向：只做多
- 资金费率估算：每日 0.005%
- 手续费：默认 taker 0.04%
- 滑点：0.05%
- 总交易成本估算：0.09%

## 杠杆规则

核心币 5x：

- BTC
- ETH
- SOL
- DOGE
- LINK

其他币 3x：

- BNB
- TRX
- XLM

注意：这里的仓位是目标名义仓位，不是保证金占用。例如 XLM 150% 名义仓位，在 3x 下约等于占用 50% 保证金。

## 入场条件

使用日线数据。

```text
日线收盘价 > 100日均线
20日均线 > 100日均线
45日价格动量 > 0
当日成交量 >= 1.5 × 20日平均成交量
```

## 风控规则

当前观察版：

- 不加固定硬止损
- 计入爆仓检查
- 爆仓近似：如果日内最低价触及杠杆对应的理论爆仓线，则损失该仓位保证金
- 每 7 天再平衡
- 组合回撤超过 15% 后，基础目标敞口降为 50%

说明：候选回测中，固定硬止损会明显降低策略表现，因此当前观察版不使用固定硬止损。
