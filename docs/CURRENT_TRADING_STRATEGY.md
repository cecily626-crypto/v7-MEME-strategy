# 当前交易策略：LBank V7-P 做多纸面账户

## 一句话版本

这是一个只做多的 LBank 纸面交易策略，用 2000 USDT 初始资金模拟运行。策略每天读取 LBank 日线数据，判断是否应该持有 BTC、ETH、SOL、XRP、DOGE、LINK；没有信号时保持空仓。

## 当前状态

- 策略方向：只做多
- 做空模块：暂不启用，后续单独开发
- 账户类型：纸面账户，不是真实下单
- 初始资金：2000 USDT
- 数据平台：LBank
- 推送渠道：新的 Telegram bot
- 当前默认交易成本：taker 0.04% + 滑点 0.05% = 0.09%
- maker 费率记录：0.02%

## 交易池

当前监控：

- BTC/USDT
- ETH/USDT
- SOL/USDT
- XRP/USDT
- DOGE/USDT
- LINK/USDT

## 入场条件

策略使用日线数据，不是小时线。

做多需要同时满足：

```text
日线收盘价 > 100日均线
20日均线 > 100日均线
45日价格动量 > 0
```

这里的 45 日动量是价格动量：

```text
当前收盘价 / 45天前收盘价 - 1
```

它不是成交量动量，也不是要求每天成交量递增。

## 仓位规则

策略会在满足条件的币种里，根据动量和波动率排序，再用波动率目标分配仓位。

默认限制：

- 最多持有 5 个币
- 单币最大权重 50%
- 最大总敞口 100%
- 每 7 天再平衡一次
- 组合回撤超过 15% 后，风险敞口降为 50%

## 为什么熊市可能空仓

因为当前版本不做空。熊市里，策略的目标是少亏或不参与，而不是从下跌中赚钱。

如果行情不满足做多条件，信号会显示：

```text
目标仓位：空仓
```

这不是系统失败，而是策略认为当前没有合适的做多机会。
