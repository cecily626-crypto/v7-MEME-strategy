# 策略：6币现货 + 放量1.2倍

## 定位

这是稳健纸账户策略，用 LBank 现货日线数据模拟运行。不使用杠杆，不做空，不计资金费率。

## 纸账户

- 初始资金：2000 USDT
- 平台数据：LBank 日线
- 方向：只做多
- 杠杆：无
- 资金费率：不计
- 手续费：默认 taker 0.04%
- 滑点：0.05%
- 总交易成本估算：0.09%

## 交易池

- BTC/USDT
- ETH/USDT
- SOL/USDT
- XRP/USDT
- DOGE/USDT
- LINK/USDT

```text
config/strategies/spot6_volume12.json
```

结果目录：

```text
results/lbank_paper/spot6_volume12/
```

## 入场条件

使用日线数据。

```text
日线收盘价 > 100日均线
20日均线 > 100日均线
45日价格动量 > 0
当日成交量 >= 1.2 × 20日平均成交量
```

45 日价格动量：

```text
当前收盘价 / 45天前收盘价 - 1
```

## 仓位规则

- 最多持有 5 个币
- 单币最大权重 50%
- 最大总敞口 100%
- 每 7 天再平衡
- 组合回撤超过 15% 后，目标敞口降为 50%

## Telegram 输出

每日信号会输出：

- 当前目标仓位
- 新增 / 加仓 / 减仓 / 清仓动作
- 如果没有变化，输出“无变化”
- 如果没有机会，输出“空仓”

日复盘和周复盘会输出：

- 当前权益
- 累计盈亏
- 最大回撤
- 交易数
- 胜率
- 费用
- 当前目标仓位
